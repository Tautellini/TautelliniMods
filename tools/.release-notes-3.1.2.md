## LockpickSettings 3.1.2

Configurable auto-solve speed.

The auto-solver's move speed is now a config value, **`autoSolveSpeed`** (in `config.lua`). This responds to feedback that the removed slow auto-solver felt more realistic: at a human pace a guard could still walk up and catch you mid-pick.

- **Default 1000** snaps each move and clears the lock in a couple of seconds (the fast behaviour from 3.1.0/3.1.1).
- **Lower it** (the vanilla baseline is 20; 40 to 80 is brisk but visible) to make the solve glide at a human-watchable pace, slow enough to get caught.

Auto-solve is still off by default and bound to F6 (solve current lock) / Shift+F6 (full-auto every lock). No other gameplay changes.

### Requirements

- Gothic 1 Remake with an **experimental UE4SS build** (the bundled `dwmapi.dll` proxy loader, placed next to `G1R-Win64-Shipping.exe`). The included readme covers this.

### Install (manual)

1. Get UE4SS working (the `complete` build on the mod host bundles it; this GitHub zip is the mod only).
2. Copy the `LockpickSettings` folder into `...\G1R\Binaries\Win64\ue4ss\Mods\`. The included `enabled.txt` activates it; no `mods.txt` editing needed.
3. Start the game and check `...\ue4ss\UE4SS.log` for the `[LockpickSettings] Loaded 3.1.2` banner.

See the bundled **`readme.txt`** for all config options (including `autoSolveSpeed`) and uninstall.
