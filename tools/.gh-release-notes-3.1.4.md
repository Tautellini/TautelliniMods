In-game settings menu support, plus extra crash hardening.

**SharedModMenu support.** With the optional SharedModMenu installed, LockpickSettings now appears as a tab there so you can change its live-tunable settings from one in-game menu instead of editing `config.lua`: the next-move hint and connection display, the full-auto toggle and auto-solve speed, and the per-tier durability bonuses (untrained / trained / master). Edits apply immediately. Built on the shared kit's cross-mod menu bridge; a no-op when SharedModMenu is not installed.

**More native-AV guards.** Extending the validity re-check 3.1.3 added for stale material handles, this build applies the same per-call `IsValid()` gating to more of the cached engine handles the adapter touches. The game can tear an object or material instance down (scene teardown / GC) between capture and a later use, and touching the dangling handle raises a native crash that `pcall` cannot catch; re-checking validity at the point of use is the only real guard.

**Internal.** The Lua test suite now runs under PUC Lua 5.4 (the runtime UE4SS actually uses) instead of LuaJIT. No shipped behaviour changes from this.

No gameplay changes. Auto-solve is still off by default: F6 solves the current lock, Shift+F6 toggles full-auto on every lock, and `autoSolveSpeed` in `config.lua` is unchanged.

Note on the occasional F6 / F7 crash: that is a known UE4SS engine bug (its deferred-action queue, RE-UE4SS issue #1180, fixed upstream in PR #1201), not a mod bug, and 3.1.4 does not work around it. A UE4SS build carrying the upstream fix resolves it.

### Requirements

- Gothic 1 Remake with an **experimental UE4SS build** (the bundled `dwmapi.dll` proxy loader, placed next to `G1R-Win64-Shipping.exe`). The bundled `readme.txt` covers this.

### Install (manual)

1. Get UE4SS working (the `complete` build on the mod host bundles it; this GitHub zip is the mod only).
2. Copy the `LockpickSettings` folder into `...\G1R\Binaries\Win64\ue4ss\Mods\`. The included `enabled.txt` activates it; no `mods.txt` editing needed.
3. Start the game and check `...\ue4ss\UE4SS.log` for the `[LockpickSettings] Loaded 3.1.4` banner.

See the bundled **`readme.txt`** for all config options and uninstall.
