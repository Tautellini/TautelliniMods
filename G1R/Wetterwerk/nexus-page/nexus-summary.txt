EXPERIMENTAL. A C++ UE4SS mod that adds a "Wetterwerk" tab to the UE4SS GUI console: switch weather presets, HOLD the weather so the game stops changing it, and live-edit the weather and sky values the game exposes (cloud, fog, rain, wind, thunder and more). Everything is driven from the menu, which opens as a separate UE4SS GUI-console window. REQUIRES UE4SS v3.0.1 Beta (Git SHA 272ce2f8) - the DLL is compiled for that exact build and will not load on a different one. No game files modified.
